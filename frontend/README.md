# LocalForge — Frontend

React + Vite + TypeScript + Tailwind CSS v3 frontend for the LocalForge AI agent.

See the root [README.md](../README.md) for full project documentation.

## Development

```bash
npm install
npm run dev       # http://localhost:5173
npm run build
npm run preview
```

## Routes

| Path | Component | Description |
|---|---|---|
| `/` | `App` → `ChatWindow` | Main chat interface |
| `/logs` | `LogsPage` | Developer log viewer (requires Developer mode in sidebar) |
